# NongusRice
Dependencies:
> ttf-droid<br>
> ttf-font-awesome<br>
> playerctl<br>
> picom or compton<br>
> nitrogen (For wallpaper)<br>
> Arch Linux (For the package count in the top bar)

# Install
1) Go into dwm, dwmblocks, and st. Run `sudo make install` for each one.
2) Add .local/bin to your path.
